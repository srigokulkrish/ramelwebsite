Job Description

We are looking for an experienced Laravel Developer to join our team. The ideal candidate should have strong expertise in PHP and Laravel, with solid experience in building and maintaining scalable web applications. The position requires a strong understanding of object-oriented programming concepts, database design, and knowledge of best practices in web development.

Key Responsibilities

  • Develop, test, and maintain web applications using Laravel.
  • Design and integrate RESTful APIs for seamless communication across systems.
  • Collaborate with front-end developers to ensure high-quality user interfaces.
  • Optimize application performance and troubleshoot issues as needed.
  • Work with databases (MySQL/PostgreSQL) to design and manage schemas effectively.
  • Use version control systems and follow agile practices with tools like JIRA.

Requirements

  • 3–5 years of professional development experience with PHP (Laravel).
  • Strong knowledge of Laravel framework and best practices.
  • Hands-on experience with RESTful API development and integration.
  • Solid understanding of JavaScript, HTML5, CSS3.
  • Proficiency in MySQL/PostgreSQL.
  • Experience with Git or other version control tools.
  • Familiarity with agile project management tools such as JIRA.
  • Experience with web services will be considered an added advantage.

Employment Details:

  • Location: Chennai
  • Type: Permanent/ 1Yr Contract (with the possibility of becoming permanent)
  • Mode of Work: Work From Office
